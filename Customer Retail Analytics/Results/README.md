# Results

This folder contains the CSV outputs generated from the SQL analysis queries.

Each file represents the result of a specific business question and serves as the data source for the Power BI dashboard. Saving query results as CSV makes the analysis reproducible and separates the SQL logic from the visualization layer.

### Folder Contents

* **Data Exploration** - Dataset overview, transaction counts, date range, and summary metrics.
* **Customer Analytics** - Customer distribution and geographic insights.
* **Sales Analytics** - Transaction trends, sales performance, and revenue metrics.
* **Product Analytics** - Best-selling and highest-revenue products.

> **Note:** These files are generated from the SQL scripts in the `/sql` folder and can be refreshed by re-running the queries after the database is updated.

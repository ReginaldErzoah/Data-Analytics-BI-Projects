# Bank Marketing Campaign Analysis

## Project Overview

This project analyzes customer data from a bank marketing campaign to identify customer segments and patterns associated with term deposit subscriptions.

The goal was to use SQL for data exploration, transformation, and analysis, then visualize key findings through a Business Intelligence dashboard.

---

## Objectives

- Explore customer demographics and campaign outcomes
- Analyze deposit subscription patterns
- Segment customers based on age and job categories
- Generate insights to support marketing decisions

---

## Tools Used

- SQL Server
- SQL
- Power BI

---

## SQL Concepts Applied

- SELECT statements
- Aggregate functions (`COUNT`)
- GROUP BY
- CASE statements
- Data transformation
- Customer segmentation

---

## Analysis Performed

- Customer distribution by deposit status
- Age group segmentation
- Job-based customer analysis
- Binary target creation for deposit subscription analysis

---

## Key Insights

- Identified customer segments with different deposit subscription behaviors
- Analyzed demographic patterns that influence campaign outcomes
- Created structured datasets for BI reporting and visualization

---

## Project Workflow

Raw Customer Data  
- SQL Exploration & Transformation  
- Analytical Tables  
- Power BI Dashboard  
- Business Insights

---

## Outcome

Developed an analytical workflow demonstrating how SQL can be used to transform raw customer data into meaningful business insights.
